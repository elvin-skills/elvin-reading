# A+100 原则

## 核心判断

A+100 不是“随便找一份比当前水平难很多的英文”。它适用于具备基础语法、愿意围绕一个窄领域持续阅读真实英文，并且有明确成果目标的使用者。

开始前确认：

1. **排他性目标**：读完这组材料后，使用者要获得什么能力或完成什么任务？如果答案只是“提高英语”，目标仍然过宽。
2. **窄领域**：材料是否来自相同主题、作者群或专业语境，使术语、概念和句式会自然复现？
3. **真实材料**：优先使用工作、研究或兴趣中本来就需要读的材料，而不是为难而难。
4. **基础语法**：使用者能否识别主干、从句、非谓语等基本结构？如果每句都在基础结构上崩溃，应先补基础或切到更易材料。
5. **可持续性**：首份材料可以慢，但每天的单位任务必须足够小，能继续，而不是一次把全文拆完。

## 为什么第一份最难

第一份同领域材料同时承担两件事：理解文本和建立领域先验。专业词、作者默认背景、常见论证方式与高频句式都还没有形成记忆，因此阻力最大。第二、第三份材料的价值不只是多读，而是让这些结构再次出现，使第一次付出的理解成本被摊薄。

因此 A+100 阅读协作的重要指标是：

- 重复术语是否被识别；
- 重复语法机制是否能用旧例解释新例；
- 旧问题是否在新证据出现时被重新调出；
- 每份新材料中“从零解释”的比例是否逐步下降。
- 相同障碍再次出现时，解释是否更短、更贴合使用者。
- 过去有效和无效的解释方式是否被后续 Agent 正确继承。

## A+1 与 A+100

- 当使用者基础薄弱、没有急迫的专业目标，或主要需要长期兴趣输入时，优先降低难度，采用 A+1。
- 当使用者已有基础、需要进入某个真实领域并希望阅读直接服务于产出时，可以采用 A+100。
- 二者可以切换。连续多次阅读都无法形成段落级理解，是材料或方法需要降阶的信号，不是意志力问题。

## 材料选择建议

第一组材料应满足：主题高度相近、长度可完成、文本可提取、有明确章节结构。尽量避免第一份就选择扫描质量差、领域跨度大、依赖大量未提供图表的材料。
